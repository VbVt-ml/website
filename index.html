About

2b2t is a minecraft server with the goal to never reset the world in a free for all no rules pvp environment, with little modification to the vanilla survival gamemode.

The world is 10+ years old, with a size of 14,198 GBs and over 733,661 players visiting at least once. The IP to connect is 2b2t.org

The server is on Minecraft Java version 1.12 and we're currently working on making it possible to update the server to 1.18. Thank you for your support.
